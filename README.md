# app-MatarMosquito
Desenvolvido em agosto de 2022. Um website com um mini game de matar mosquito.
Ao iniciar pode escolher a dificuldade do jogo, existindo três dificuldades, normal-difícil-Chuck Norris.
Para ganhar o jogo, precisa matar os todos os mosquitos que aparecer aleatoriamente na tela durante 15 segundos, a cada mosquito que passar vivo, perderá uma vida, tendo três no total.
